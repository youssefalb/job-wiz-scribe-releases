# Job Wiz Scribe - Desktop App Releases

Official releases for the Job Wiz Scribe Desktop Interview Assistant.

## Downloads
Download the latest version from the [Releases](https://github.com/youssefalb/job-wiz-scribe-releases/releases) page.

### Installation

**Windows:**
1. Download `Interview-Assistant-Setup-{version}-Windows.exe`
2. Run the installer
3. Windows Defender may show a warning (app is not code-signed)
4. Click "More info" → "Run anyway"

**macOS:**
1. Download `Interview-Assistant-{version}-macOS.dmg`
2. Open the DMG file
3. Drag Interview Assistant to Applications
4. First launch: Right-click → Open (bypass Gatekeeper)

## Features
- Real-time interview transcription
- AI-powered interview assistance
- Dual audio capture (microphone + system audio)
- Transparent, always-on-top UI

## System Requirements
- Windows 10+ (x64) or macOS 10.13+
